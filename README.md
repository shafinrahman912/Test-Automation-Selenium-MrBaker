### Prerequisites

Required software to run the project.

```
- jdk 17 or above
- Apache Maven 3.8.x
- Eclipse or IntelliJ IDE ( recommended for development and debugging purposes).
- Selenium WebDriver
- TestNG
- Maven Repositories
```
